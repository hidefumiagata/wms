# フロントエンドアーキテクチャ

## 技術スタック

| 役割 | 採用技術 | バージョン方針 |
|------|---------|-------------|
| **フレームワーク** | Vue 3（Composition API） | 最新安定版 |
| **言語** | TypeScript | 最新安定版 |
| **UIライブラリ** | Element Plus | 最新安定版 |
| **ビルドツール** | Vite | 最新安定版 |
| **状態管理** | Pinia | 最新安定版 |
| **ルーティング** | Vue Router | 最新安定版 |
| **HTTPクライアント** | Axios | 最新安定版 |
| **フォームバリデーション** | VeeValidate + Zod | 最新安定版 |
| **多言語対応** | vue-i18n | 最新安定版 |
| **単体テスト** | Vitest | 最新安定版 |
| **E2Eテスト** | Playwright | 最新安定版 |

## 画面レイアウト

**サイドメニュー型** を採用する。

```
┌─────────────────────────────────────────┐
│  [ロゴ / システム名]  [ユーザ情報] [ログアウト]  │  ← ヘッダー
├──────────┬──────────────────────────────┤
│          │  パンくずリスト                │
│ サイド   │ ─────────────────────────── │
│ メニュー  │                              │
│          │  メインコンテンツエリア         │
│ ・入荷    │                              │
│ ・在庫    │                              │
│ ・出荷    │                              │
│ ・マスタ  │                              │
│ ・レポート │                              │
│ ・バッチ  │                              │
└──────────┴──────────────────────────────┘
```

## 多言語対応

- **vue-i18n** を使用
- 初期対応言語：日本語（ja）・英語（en）
- 言語ファイル：`src/locales/ja.json`、`src/locales/en.json`
- デフォルト言語：日本語
- 言語切替：ヘッダーから選択可能

## ディレクトリ構成（案）

```
frontend/
├── src/
│   ├── assets/          # 静的ファイル
│   ├── components/      # 共通コンポーネント
│   ├── composables/     # Composition API（共通ロジック）
│   ├── layouts/         # レイアウトコンポーネント
│   ├── locales/         # i18n言語ファイル
│   │   ├── ja.json
│   │   └── en.json
│   ├── router/          # Vue Router設定
│   ├── stores/          # Piniaストア
│   ├── types/           # TypeScript型定義
│   ├── utils/           # ユーティリティ関数
│   ├── views/           # ページコンポーネント
│   │   ├── inbound/     # 入荷管理画面
│   │   ├── inventory/   # 在庫管理画面
│   │   ├── outbound/    # 出荷管理画面
│   │   ├── master/      # マスタ管理画面
│   │   ├── report/      # レポート画面
│   │   └── batch/       # バッチ実行画面
│   ├── App.vue
│   └── main.ts
├── tests/
│   ├── unit/            # Vitestユニットテスト
│   └── e2e/             # Playwrightテスト
├── index.html
├── vite.config.ts
├── tsconfig.json
└── package.json
```

## API通信

- Axios インスタンスを共通設定（`src/utils/api.ts`）
- リクエストインターセプター：JWTトークンを `Authorization: Bearer` ヘッダーに自動付与
- レスポンスインターセプター：401エラー時にログイン画面へリダイレクト
