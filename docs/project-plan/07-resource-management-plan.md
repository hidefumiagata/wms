# リソース管理計画 (Resource Management Plan)

## 1. プロジェクト体制

| 役割 | 担当 |
|------|------|
| **プロジェクトオーナー / PM / レビュアー** | プロジェクトオーナー |
| **開発者** | プロジェクトオーナー ＋ Claude Code |

## 2. 開発環境

| 項目 | 内容 |
|------|------|
| **開発マシン** | WSL2（Ubuntu on Windows） |
| **ローカル実行環境** | Docker Compose（DB・依存サービスをローカルで起動） |
| **ソースコード管理** | Git / GitHub |
| **IDE** | VSCode（Claude Code拡張含む） |
