3. コスト削減施策

PostgreSQL 停止運用

未使用時（開発・デモ以外）はPostgreSQL Flexible Serverを手動停止する。


操作コマンド
停止az postgres flexible-server stop --resource-group <rg> --name <server>
起動az postgres flexible-server start --resource-group <rg> --name <server>

⚠️ Azure Flexible Serverは7日間連続停止で自動再起動。長期未使用時は定期的な手動停止が必要。

開発環境の運用方針


項目方針
dev環境開発・テスト時のみ起動。未使用時はDB停止
prd環境ShowCase公開期間のみ常時起動。それ以外はDB停止
Container Appsmin replicas=0 のため自動ゼロスケール。停止操作不要
