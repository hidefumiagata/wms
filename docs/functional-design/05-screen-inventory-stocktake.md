## INV-011 棚卸一覧

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INV-011` |
| **画面名** | 棚卸一覧 |
| **URL パス** | `/inventory/stocktakes` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF, VIEWER |
| **概要** | 実施済み・実施中の棚卸ヘッダー一覧を表示する。棚卸IDや実施日、対象エリア、ステータスで絞り込み検索ができ、各棚卸の詳細（実数入力・確定）画面へ遷移できる。WAREHOUSE_MANAGERは新規棚卸の開始ができる。 |

- 検索フォームエリア: 棚卸ID・実施日範囲・対象棟・ステータスで絞り込み

## INV-012 棚卸開始

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INV-012` |
| **画面名** | 棚卸開始 |
| **URL パス** | `/inventory/stocktakes/new` |
| **対象ロール** | WAREHOUSE_MANAGER |
| **概要** | 棚卸の対象ロケーション範囲（棟・エリア）を指定して棚卸を開始するフォーム画面。棟・エリアを選択すると対象ロケーション数がプレビュー表示される。棚卸開始と同時に対象ロケーションに棚卸ロックがかかる。棚卸リスト（RPT-010）の帳票出力も本画面から実行できる。 |

**コンテンツエリアの詳細**

- 画面上部に警告バナー「棚卸中のロケーションへの入出庫は制限されます。開始前に入出庫作業が完了していることを確認してください。」
- 棟・エリアを選択すると対象ロケーション数を非同期で取得してプレビューカードに表示
- 「棚卸リスト出力」: 棚卸開始前でも選択したロケーション範囲でリスト出力可
- 「棚卸開始」クリック → 確認ダイアログ表示 → OK で API 呼び出し → INV-013 へ遷移

## INV-013 棚卸実施（実数入力）

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INV-013` |
| **画面名** | 棚卸実施（実数入力） |
| **URL パス** | `/inventory/stocktakes/:id` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF |
| **概要** | 棚卸対象のロケーション×商品×荷姿ごとに「棚卸前在庫数（読み取り専用）」と「実数（入力欄）」を並べて表示し、担当者が実際に数えた数量を入力する画面。ロケーションコード順に表示。一時保存と棚卸リスト出力が可能。 |

**コンテンツエリアの詳細**

- ヘッダー情報: 棚卸ID・対象棟/エリア・実施日・ステータスをサマリーバーで表示
- ツールバー: 「棚卸リスト出力」ボタン（RPT-010）、未入力件数カウンター
- 入力テーブル: ロケーションコード順に表示。棚卸前在庫は読み取り専用。実数欄は数値入力（0以上の整数）
- 入力状態インジケーター: 未入力（空欄）/ 差異あり（実数≠棚卸前在庫）を視覚表示
- ロケーションが変わる行に区切り線を表示
- 「一時保存」: 入力途中の実数を保存。画面を閉じて再開可能
- 「確定へ」: 全項目入力済みの場合のみ活性。INV-014へ遷移
