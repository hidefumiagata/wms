## ALL-001 在庫引当

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `ALL-001` |
| **画面名** | 在庫引当 |
| **URL パス** | `/allocation` |
| **対象ロール** | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| **概要** | 受注に対する在庫引当を実行・管理する画面。「引当実行」タブでは未引当の受注を検索・選択して一括引当を実行し、ばらし指示の確認・完了操作を行う。「引当済み一覧」タブでは引当済みの受注を確認し、必要に応じて引当を解放する。 |

コンテンツエリアの構成:
1. **タブバー**: 「引当実行」「引当済み一覧」の2タブ切替
2. **引当実行タブ**: 検索フォーム + 受注一覧テーブル（チェックボックス付き） + 引当実行ボタン + 引当結果エリア

### 4. イベント一覧

| イベントID | トリガー | 処理概要 | 遷移先 / 結果 | 実行可能ロール |
|----------|---------|---------|-------------|-------------|
| `EVT-ALL001-001` | 画面初期表示 | 引当実行タブをアクティブにし、ステータス「受注/一部引当」で引当対象受注一覧を取得（GET /api/v1/allocation/orders） | — | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-002` | 「検索する」ボタンクリック | 検索条件を元に引当対象受注一覧を再取得 | 一覧テーブル更新 | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-003` | 「クリア」ボタンクリック | 検索条件を初期状態にリセット | 検索フォームクリア | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-004` | 「選択した受注を引当実行」ボタンクリック | 選択された受注IDリストで POST /api/v1/allocation/execute を呼び出し。結果を引当結果エリアに表示 | 引当結果エリア表示 | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-005` | ばらし指示の「完了」ボタンクリック | PUT /api/v1/allocation/unpack-instructions/{id}/complete を呼び出し。成功後ステータスをCOMPLETEDに更新 | 同画面（ばらし指示ステータス更新） | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-006` | 「引当済み一覧」タブ切替 | GET /api/v1/allocation/allocated-orders で引当済み受注一覧を取得 | 引当済み一覧タブ表示 | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-007` | 「引当解放」ボタンクリック | 引当解放確認ダイアログを表示 | ダイアログ表示 | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| `EVT-ALL001-008` | 確認ダイアログ「引当解放する」クリック | POST /api/v1/allocation/release を呼び出し。成功後に引当済み一覧を再取得し、成功メッセージ表示 | 引当済み一覧タブ更新 | SYSTEM_ADMIN, WAREHOUSE_MANAGER |

### 5. メッセージ一覧

| メッセージID | 種別 | 発生条件 | メッセージ文（日本語） |
|------------|------|---------|-------------------|
| `MSG-E-ALL001-001` | エラー | 引当実行ボタン押下時に受注が未選択 | 引当実行する受注を1件以上選択してください |
| `MSG-E-ALL001-002` | エラー | 引当実行APIが400エラーを返した場合 | 引当実行に失敗しました: {errorMessage} |
| `MSG-E-ALL001-003` | エラー | 引当解放APIが409エラーを返した場合 | ピッキング指示済み以降の受注は引当解放できません |
| `MSG-E-ALL001-004` | エラー | 引当解放APIが400/404エラーを返した場合 | 引当解放に失敗しました: {errorMessage} |
| `MSG-W-ALL001-001` | 警告 | 引当解放ボタンクリック時 | 受注番号「{slipNumber}」の引当を解放します。引当済み在庫が元に戻されます。よろしいですか？ |
| `MSG-S-ALL001-001` | 成功 | 引当実行成功 | 引当を実行しました（成功: {successCount}件） |
| `MSG-S-ALL001-002` | 成功 | ばらし完了成功 | ばらし指示を完了しました |
| `MSG-S-ALL001-003` | 成功 | 引当解放成功 | 引当を解放しました |
| `MSG-I-ALL001-001` | 情報 | 引当実行結果に未引当明細がある場合 | 在庫不足のため一部の明細が引当できませんでした |
