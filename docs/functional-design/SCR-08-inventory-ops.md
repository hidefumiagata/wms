## INV-001 在庫一覧照会

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INV-001` |
| **画面名** | 在庫一覧照会 |
| **URL パス** | `/inventory` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF, VIEWER |
| **概要** | ロケーション・商品・荷姿・保管条件で在庫を絞り込んで一覧表示する。ロケーション別明細表示と商品単位の合計在庫数（荷姿別）の切り替えが可能。在庫一覧・在庫推移レポートをCSV/PDF出力できる。 |

**コンテンツエリア詳細**

#### 検索フォーム

| 項目 | 配置 |
|------|------|
| ロケーション（前方一致） | 1行目左 |
| 商品（セレクト） | 1行目中 |
| 荷姿（セレクト） | 1行目右 |
| 保管条件（セレクト） | 2行目左 |
| 検索・クリアボタン | 2行目右端 |

#### 表示切替タブ

- **ロケーション別（デフォルト）**: ロケーション × 商品 × 荷姿 の1行1レコード表示
- **商品合計**: 商品ごとに荷姿別合計在庫数を集計して表示

#### ロケーション別テーブル列

| 列 | 内容 |
|----|------|
| # | 行番号 |
| ロケーション | ロケーションコード（例: A-01-A-01-01-01） |
| 保管条件 | 常温 / 冷蔵 / 冷凍 |
| 商品コード | |
| 商品名 | |
| 荷姿 | ケース / ボール / バラ |
| 在庫数 | 数量（右寄せ） |
| 引当数 | 引当済み数量（右寄せ）。0の場合はハイフン表示 |
| 有効在庫数 | 在庫数 − 引当数（右寄せ） |
| 操作 | [移動] ボタン → INV-002へ遷移（VIEWERには非表示） |

> **備考**: 有効在庫数 = 在庫数 - 引当数。引当数 > 0 の行は視覚的に区別する（背景色やアイコン）。

#### 商品合計テーブル列

| 列 | 内容 |
|----|------|
| 商品コード | |
| 商品名 | |
| ケース合計 | 全ロケーションのケース在庫合計 |
| ボール合計 | 全ロケーションのボール在庫合計 |
| バラ合計 | 全ロケーションのバラ在庫合計 |
| 合計（バラ換算） | ケース・ボール・バラをバラ単位に換算した総数 |

## INV-002 在庫移動登録

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INV-002` |
| **画面名** | 在庫移動登録 |
| **URL パス** | `/inventory/move` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF |
| **概要** | 在庫を移動元ロケーションから移動先ロケーションへ移動する。商品・荷姿・移動数量を指定して登録することで在庫レコードを更新する。ロケーション収容制約・在庫マイナス禁止ルールをバリデーションする。 |
