## MST-031 棟一覧

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-031` |
| **画面名** | 棟一覧 |
| **URL パス** | `/master/buildings` |
| **対象ロール** | SYSTEM_ADMIN |
| **概要** | 選択中倉庫に登録されている棟の一覧を表示する。有効/無効の絞り込みができ、新規登録・編集・有効/無効切り替えを行う。 |

```
┌─────────────────────────────────────────────────────────────┐

│  商品    │──────────────────────────────────────────────────│

│  倉庫   │──────────────────────────────────────────────────│
│ ▶ 棟    │ 検索フォーム                                       │
│  エリア  │  有効/無効: [すべて▼]          [検索] [クリア]     │
│  ロケー  │──────────────────────────────────────────────────│
│  ション  │ 件数: 4件                                         │
│  ユーザー│──────────────────────────────────────────────────│

│          │   │          │        │            │      │[無効化]│
└──────────┴──────────────────────────────────────────────────┘
                              ← 1 →   20件/ページ▼
```

コンテンツエリア構成:
- **検索フォームエリア**: 有効/無効フィルタ（セレクト）
- **ツールバー**: 左側に「＋ 棟登録」ボタン、右側に件数表示
- **一覧テーブル**: 棟コード・棟名称・倉庫コード・状態・操作列
- **操作列**: 「編集」ボタン（常時表示）、「無効化」または「有効化」ボタン（状態に応じて切り替え）
- **ページネーション**: テーブル下部
