## MST-061 ユーザー一覧

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-061` |
| **画面名** | ユーザー一覧 |
| **URL パス** | `/master/users` |
| **対象ロール** | SYSTEM_ADMIN |
| **概要** | システムに登録されている全ユーザーを一覧表示する。氏名・ロール・ステータスで絞り込み検索が可能。アカウントロック中ユーザーは視覚的に識別でき、ロック解除操作への導線を提供する。 |

---

### 2. 画面レイアウト

**モックアップ**: [MST-061-user-list.html](mockups/MST-061-user-list.html)

```

│            │ ホーム › マスタ管理 › ユーザー一覧                   │

│ ・マスタ管理│────────────────────────────────────────────────────│

│  （active） │  [氏名/ユーザーコード]  [ロール▼]  [ステータス▼]    │
│            │                               [🔍 検索] [クリア]   │

│            │  全 NN 件                                           │
│            │────────────────────────────────────────────────────│

│            │             ← 1 2 3 →       20件/ページ▼           │
└────────────┴────────────────────────────────────────────────────┘
```

**コンテンツエリア詳細**:

- **検索フォーム**: 氏名またはユーザーコード（部分一致）、ロール（全て/SYSTEM_ADMIN/WAREHOUSE_MANAGER/WAREHOUSE_STAFF/VIEWER）、ステータス（全て/有効/無効/ロック中）
- **一覧テーブル**: ユーザーコード・氏名・メールアドレス・ロール・ステータス・操作列
- **ロック中表示**: ステータス列に🔒アイコン付き赤バッジ「ロック中」を表示。行全体を薄い赤背景で強調
- **操作列**: [編集] ボタン常時表示。ロック中ユーザーのみ [ロック解除] ボタンを追加表示
- **無効ユーザーの表示**: ステータス列にグレーバッジ「無効」を表示。行テキストを薄色（gray-400）で表示

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-063` |
| **画面名** | ユーザー編集 |
| **URL パス** | `/master/users/:id/edit` |
| **対象ロール** | SYSTEM_ADMIN |
| **概要** | 登録済みユーザーの情報（氏名・メールアドレス・ロール・有効/無効）を更新する。ユーザーコードは読み取り専用。アカウントロック中の場合はロック解除ボタンを表示する。自分自身のロール変更・無効化は不可。 |
