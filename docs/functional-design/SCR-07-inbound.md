## INB-001 入荷予定一覧

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INB-001` |
| **画面名** | 入荷予定一覧 |
| **URL パス** | `/inbound/slips` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF, VIEWER |
| **概要** | 登録済みの入荷予定伝票を一覧表示する。ステータス・入荷予定日・仕入先での絞り込み、入荷予定登録への遷移、および入荷予定レポート（RPT-03）のPDF出力を行う画面。 |

1. **ページタイトル＋新規登録ボタン**: 右上に「＋ 新規登録」ボタン（VIEWER非表示）

3. **ツールバー**: 件数表示・入荷予定レポート出力ボタン
4. **一覧テーブル**: 伝票番号・種別・入荷予定日・仕入先・明細数・ステータスを表示。行クリックで詳細へ遷移
5. **ページネーション**: 20件/ページ（変更可）

## INB-002 入荷予定登録

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `INB-002` |
| **画面名** | 入荷予定登録 |
| **URL パス** | `/inbound/slips/new` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF |
| **概要** | 通常入荷の入荷予定を手動で登録する画面。入荷予定日・仕入先などのヘッダー情報と、商品・予定数量の明細を入力して保存する。登録時のステータスは「入荷予定」で固定される。 |

2. **ツールバー**: 件数表示・入庫実績レポート出力ボタン
3. **実績テーブル**: 伝票番号・入庫完了日・仕入先・商品コード・商品名・予定数量・検品数・差異数を表示。差異数が0以外の行は差異数を赤字表示
4. **ページネーション**: 20件/ページ（変更可）
