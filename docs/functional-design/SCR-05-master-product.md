## MST-001 商品一覧

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-001` |
| **画面名** | 商品一覧 |
| **URL パス** | `/master/products` |
| **対象ロール** | SYSTEM_ADMIN, WAREHOUSE_MANAGER |
| **概要** | 登録済み商品の一覧を表示する。商品コード・商品名・保管条件・有効/無効で絞り込みができ、商品の新規登録・編集・有効/無効切替を行うエントリポイントとなる画面。 |

---

### 2. 画面レイアウト

**モックアップ**: [MST-001-product-list.html](mockups/MST-001-product-list.html)

```
┌─────────────────────────────────────────────────────────────────┐
│ [メタバー] MST-001 | /master/products | SYSTEM_ADMIN            │
├─────────────────────────────────────────────────────────────────┤
│ ヘッダー（WMS / 倉庫切替▼ / ユーザー名 / ログアウト）           │
├──────────┬──────────────────────────────────────────────────────┤
│ マスタ管理│ ホーム › マスタ管理 › 商品一覧                       │
│  > 商品  │──────────────────────────────────────────────────────│
│  取引先  │ 商品一覧                   [＋ 新規登録]              │
│  倉庫    │──────────────────────────────────────────────────────│
│  棟      │ [検索フォーム]                                         │
│  エリア  │  商品コード [____]  商品名 [________]                   │
│  ロケ    │  保管条件 [▼]  有効/無効 [▼]  [検索] [クリア]        │
│  ユーザー│──────────────────────────────────────────────────────│
│          │ [全 XX 件]                                            │
│          │──┬──────────┬──────────┬──────┬──────┬────┬──────────│
│          │# │商品コード │商品名    │カテゴリ│保管条件│有効│操作  │
│          │──┼──────────┼──────────┼──────┼──────┼────┼──────────│
│          │1 │...       │...       │...   │...   │●  │[編集][無効化]│
│          │──┴──────────┴──────────┴──────┴──────┴────┴──────────│
│          │            ← 1 2 3 ... 10 →   20件/ページ▼           │
└──────────┴──────────────────────────────────────────────────────┘
```

**検索フォームエリア**:
- 商品コード（前方一致テキスト入力）
- 商品名（部分一致テキスト入力）
- 保管条件（セレクト: すべて / 常温 / 冷蔵 / 冷凍）
- 有効/無効（セレクト: すべて / 有効のみ / 無効のみ）
- [検索] ボタン / [クリア] ボタン

**テーブルエリア**:
- 件数表示: `全 XX 件`
- カラム: #（連番）/ 商品コード / 商品名 / 商品名カナ / 保管条件 / 危険物 / ロット管理 / 有効/無効 / 操作
- 操作列: [編集] ボタン / [無効化] または [有効化] ボタン

**ページングエリア**:
- ページ番号ボタン / 前・次ボタン
- 表示件数切替: 20件 / 50件 / 100件

## MST-002 商品登録

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-002` |
| **画面名** | 商品登録 |
| **URL パス** | `/master/products/new` |

| **概要** | 新規商品を登録するフォーム画面。商品コード・商品名・保管条件・各種フラグ等の全管理項目を入力して商品を新規作成する。 |

### 2. 画面レイアウト

**モックアップ**: [MST-002-product-new.html](mockups/MST-002-product-new.html)

```
┌─────────────────────────────────────────────────────────────────┐

│          │──────────────────────────────────────────────────────│
│          │ 商品登録                                               │
│          │──────────────────────────────────────────────────────│
│          │ [基本情報]                                             │
│          │  商品コード *  [__________]  商品名 *  [__________]  │
│          │  商品名カナ *  [__________]  バーコード [__________]  │
│          │──────────────────────────────────────────────────────│
│          │ [保管・管理設定]                                       │
│          │  保管条件 *   ○常温 ○冷蔵 ○冷凍                    │
│          │  ケース入数   [____]        ボール入数 [____]         │
│          │──────────────────────────────────────────────────────│
│          │ [フラグ設定]                                           │
│          │  □危険物フラグ  □ロット管理フラグ                    │
│          │  □賞味/使用期限管理フラグ  □出荷禁止フラグ           │

│          │                        [キャンセル]  [登録]           │
└──────────┴──────────────────────────────────────────────────────┘
```

**フォームセクション構成**:

**セクション1: 基本情報**
- 商品コード（必須）/ 商品名（必須）: 2列レイアウト
- 商品名カナ（必須）/ バーコード/JANコード: 2列レイアウト

**セクション2: 保管・管理設定**
- 保管条件（必須）: ラジオボタン 常温 / 冷蔵 / 冷凍
- ケース入数 / ボール入数: 2列レイアウト（数値入力）

**セクション3: フラグ設定**
- 危険物フラグ: チェックボックス
- ロット管理フラグ: チェックボックス
- 賞味/使用期限管理フラグ: チェックボックス
- 出荷禁止フラグ: チェックボックス
- 有効/無効フラグ: ラジオボタン（デフォルト: 有効）

**フッターボタン**:
- [キャンセル] — MST-001へ戻る
- [登録] — バリデーション後に商品を登録

## MST-003 商品編集

| 項目 | 内容 |
|------|------|
| **画面ID** | `MST-003` |
| **画面名** | 商品編集 |
| **URL パス** | `/master/products/:id/edit` |
