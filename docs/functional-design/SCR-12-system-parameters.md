## SYS-001 システムパラメータ設定

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `SYS-001` |
| **画面名** | システムパラメータ設定 |
| **URL パス** | `/system/parameters` |
| **対象ロール** | SYSTEM_ADMIN |
| **概要** | システム全体の動作を制御するパラメータの一覧表示・編集画面。カテゴリ別にグルーピングされたテーブル形式で表示し、各パラメータの値をインライン編集できる。 |

---

### 2. 画面レイアウト

```
┌─────────────────────────────────────────────────────────────────┐
│ ヘッダー（WMS ロゴ / 倉庫切替▼ / ログイン中: 管理者 / ログアウト） │
├────────────┬────────────────────────────────────────────────────┤
│            │ ホーム › システム管理 › システムパラメータ設定         │
│ サイドバー  │ システムパラメータ設定                                │
│ ・システム  │────────────────────────────────────────────────────│
│  > パラメータ│ ▼ INVENTORY（在庫管理）                              │
│  （active） │──────────────────────────────────────────────────── │
│            │ パラメータ名       │ 現在値  │ デフォルト │ 説明   │操作│
│            │────────────────────┼────────┼──────────┼───────┼───│
│            │ LOCATION_CAPACITY  │ [10  ] │ 1        │ 1ロケ...│[保存]│
│            │ STOCK_THRESHOLD    │ [100 ] │ 50       │ 在庫...  │[保存]│
│            │────────────────────────────────────────────────────│
│            │ ▼ OUTBOUND（出庫管理）                                │
│            │──────────────────────────────────────────────────── │
│            │ パラメータ名       │ 現在値  │ デフォルト │ 説明   │操作│
│            │────────────────────┼────────┼──────────┼───────┼───│
│            │ PICKING_BATCH_SIZE │ [50  ] │ 30       │ ピッキ...│[保存]│
│            │────────────────────────────────────────────────────│
└────────────┴────────────────────────────────────────────────────┘
```

**コンテンツエリア詳細**:

- **カテゴリグルーピング**: パラメータは `category` フィールドでグルーピングし、カテゴリ名をセクションヘッダーとして表示する。カテゴリは折りたたみ可能（デフォルトは展開状態）
- **インライン編集**: 「現在値」列は直接編集可能な入力フィールドとして表示する。値の型（`valueType`）に応じて入力コントロールを切り替える（INTEGER: 数値入力、STRING: テキスト入力、BOOLEAN: トグルスイッチ）
- **保存ボタン**: 各行に個別の保存ボタンを配置する。値が変更されていない場合は disabled 状態とする
- **デフォルト値表示**: 変更の参考として、デフォルト値を読み取り専用で表示する

### 3. 画面項目一覧

| 項目ID | 項目名 | 種別 | 必須 | 初期値 | バリデーション | 備考 |
|--------|--------|------|:----:|--------|--------------|------|
| LST-001 | カテゴリ | 表示 | — | — | — | グルーピングヘッダー。折りたたみ可能 |
| LST-002 | パラメータ名 | 表示 | — | — | — | `displayName` を表示。ツールチップで `paramKey` を表示 |
| LST-003 | 現在値 | インライン編集 | ○ | 登録済み値 | `valueType` に応じたバリデーション | INTEGER: 1以上の整数、STRING: 最大500文字、BOOLEAN: true/false |
| LST-004 | デフォルト値 | 表示 | — | — | — | 読み取り専用 |
| LST-005 | 説明 | 表示 | — | — | — | `description` フィールドの内容を表示 |
| LST-006 | 最終更新日時 | 表示 | — | — | — | yyyy/MM/dd HH:mm。ツールチップで更新者名を表示 |
| BTN-001 | 保存 | ボタン | — | — | — | 各行に配置。値未変更時は disabled |

### 4. イベント一覧

| イベントID | トリガー | 処理概要 | 遷移先 / 結果 | 実行可能ロール |
|----------|---------|---------|-------------|-------------|
| `EVT-SYS-001` | 画面表示 | `GET /api/v1/system/parameters` でパラメータ一覧を取得し、カテゴリ別にグルーピング表示 | 同画面（データ表示） | SYSTEM_ADMIN |
| `EVT-SYS-002` | 現在値フィールド変更 | 入力値のフロントエンドバリデーションを実行。値が変更された行の [保存] ボタンを活性化 | 同画面（保存ボタン活性化） | SYSTEM_ADMIN |
| `EVT-SYS-003` | [保存] ボタンクリック | 確認ダイアログ表示 → OK で `PUT /api/v1/system/parameters/{paramKey}` を呼び出し | 同画面。成功メッセージ表示。該当行の値を更新後の値で再表示 | SYSTEM_ADMIN |
| `EVT-SYS-004` | カテゴリヘッダークリック | カテゴリセクションの折りたたみ/展開をトグル | 同画面（表示切替） | SYSTEM_ADMIN |
