## OUT-022 出荷完了（配送情報入力）

### 1. 画面概要

| 項目 | 内容 |
|------|------|
| **画面ID** | `OUT-022` |
| **画面名** | 出荷完了（配送情報入力） |
| **URL パス** | `/outbound/slips/:id/ship` |
| **対象ロール** | WAREHOUSE_MANAGER, WAREHOUSE_STAFF |
| **概要** | 出荷検品完了後に出荷を確定し、配送業者・送り状番号を入力する画面。確定時に引当在庫が実在庫から減算され、ステータスが「出荷完了」に遷移する。 |
