## テストデータ前提

全テストシナリオで以下のテストデータがFlywayスクリプトにより投入済みであることを前提とする。

| データ種別 | 説明 |
|----------|------|
| ユーザー | `admin001`（SYSTEM_ADMIN）、`mgr001`（WAREHOUSE_MANAGER）、`staff001`（WAREHOUSE_STAFF）、`viewer001`（VIEWER） |
| 倉庫 | `WARA`（東京DC、有効）、`WARB`（大阪DC、有効） |
| 棟 | `A`（WARA配下、有効）、`B`（WARA配下、有効） |
| エリア | `01-A`（棟A配下、常温・在庫エリア、有効）、`INBOUND`（棟A配下、常温・入荷エリア、有効） |
| ロケーション | `A-01-A-01-01-01`（エリア01-A配下、有効）、入荷エリア用ロケーション1件 |
| 商品 | `PRD-001`（常温、ロット管理OFF、有効）、`PRD-002`（冷蔵、ロット管理ON、有効） |
| 取引先 | `SUP-001`（仕入先、有効）、`CUS-001`（出荷先、有効） |

### SC-C09: 楽観ロック競合テスト（結合テスト例）

```typescript
test('SC-C09: 楽観ロック競合で409が返される', async ({ request }) => {
  // 前提: 商品を登録
  const createRes = await request.post('/api/v1/master/products', {
    data: {
      productCode: `TST-OPT-${Date.now()}`,
      productName: 'ロックテスト商品',
      productNameKana: 'ロックテストショウヒン',
      caseQuantity: 1,
      ballQuantity: 1,
      storageCondition: 'AMBIENT',
      isHazardous: false,
      lotManageFlag: false,
      expiryManageFlag: false,
      shipmentStopFlag: false,
      isActive: true,
    },
  });
  const product = await createRes.json();

  // ユーザーBが先に更新（version 0 → 1）
  const updateB = await request.put(`/api/v1/master/products/${product.id}`, {
    data: {
      productName: 'ユーザーBの更新',
      productNameKana: 'ユーザービーノコウシン',
      caseQuantity: 1,
      ballQuantity: 1,
      storageCondition: 'AMBIENT',
      isHazardous: false,
      lotManageFlag: false,
      expiryManageFlag: false,
      shipmentStopFlag: false,
      isActive: true,
      version: 0, // 取得時のversion
    },
  });
  expect(updateB.status()).toBe(200);

  // ユーザーAが古いversionで更新を試みる
  const updateA = await request.put(`/api/v1/master/products/${product.id}`, {
    data: {
      productName: 'ユーザーAの更新',
      productNameKana: 'ユーザーエーノコウシン',
      caseQuantity: 1,
      ballQuantity: 1,
      storageCondition: 'AMBIENT',
      isHazardous: false,
      lotManageFlag: false,
      expiryManageFlag: false,
      shipmentStopFlag: false,
      isActive: true,
      version: 0, // 古いversion → 競合
    },
  });
  expect(updateA.status()).toBe(409);
  const errorBody = await updateA.json();
  expect(errorBody.errorCode).toBe('OPTIMISTIC_LOCK_CONFLICT');
});
```
